# Readynest Task 5

A real-time Multi-tenant SaaS platform built with React, Vite, Node.js, TypeScript, and WebSockets.

## Project Structure
* **frontend/** - React/Vite client dashboard and Kanban board.
* **backend/** - Express server managing workspace database states and socket channels.

## Setup & Installation

### Backend Setup
1. Navigate to the backend directory: `cd backend`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

### Frontend Setup
1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
